## Como Abrir

### Opção 1: Duplo clique
1. Navegue até a pasta `Olokun`
2. Localize o arquivo `index.html`
3. Faça duplo clique para abrir no navegador padrão

### Opção 2: Arrastar para o navegador
1. Abra seu navegador (Chrome, Firefox, Edge, Safari)
2. Arraste o arquivo `index.html` para a janela do navegador

### Opção 3: Via linha de comando
```bash
# Windows
start index.html

# macOS
open index.html

# Linux
xdg-open index.html
```

### Opção 4: Desde VS Code
1. Clique com botão direito em `index.html`
2. Selecione "Open with Live Server" (se tiver a extensão instalada)
3. Ou pressione `Alt + L` + `Alt + O`

## Navegação

### 🖱️ Navegação com Mouse
- **Clique** nos botões das abas para trocar de conteúdo
- **Hover** sobre os cards para ver efeitos de sombra e movimento
- Todos os botões e links são responsivos

### ⌨️ Navegação com Teclado
- **Seta Esquerda (←)**: Vai para a aba anterior
- **Seta Direita (→)**: Vai para a próxima aba
- **Tab**: Navega entre elementos interativos
- **Enter**: Ativa o elemento focado
- **Esc**: Fecha modais (como o vídeo)

### 📱 Navegação em Mobile
- **Toque** nos botões das abas
- **Deslize** para cima/baixo para ver mais conteúdo
- O design se adapta automaticamente para telas pequenas

## Resolução de Problemas

### ❓ As abas não estão funcionando

**Solução 1:** Verifique se o JavaScript está habilitado
- Chrome: Menu → Configurações → Privacidade → JavaScript (habilitado)
- Firefox: about:config → javascript.enabled (true)

**Solução 2:** Recarregue a página
- Pressione `F5` ou `Ctrl+R` (Windows/Linux)
- Pressione `Cmd+R` (macOS)

**Solução 3:** Limpe o cache
- Pressione `Ctrl+Shift+Del` (Windows/Linux)
- Pressione `Cmd+Shift+Del` (macOS)

---

### ❓ O design está estranho / desalinhado

**Solução:** O CSS pode não estar carregando corretamente
- Verifique se o arquivo `styles.css` está no mesmo diretório
- Recarregue a página (Ctrl+R)
- Abra em um navegador diferente para testar

---

### ❓ Os vídeos não estão tocando

**Solução:** Verifique a conexão com a Internet
- O vídeo está hospedado no YouTube
- Certifique-se de ter conexão com internet estável
- Se usar VPN, tente desativar temporariamente

---

### ❓ O layout está muito pequeno / muito grande

**Solução:** Ajuste o zoom do navegador
- **Zoom in:** `Ctrl + +` (Windows/Linux) ou `Cmd + +` (macOS)
- **Zoom out:** `Ctrl + -` (Windows/Linux) ou `Cmd + -` (macOS)
- **Reset:** `Ctrl + 0` (Windows/Linux) ou `Cmd + 0` (macOS)

---

### ❓ A navegação por teclado não funciona

**Solução:** Certifique-se de que o foco está na página
- Clique em qualquer lugar da página para focar
- Então use as setas esquerda/direita para navegar
- Pressione `Tab` para navegar entre elementos

## 📞 Suporte

Se encontrar problemas não listados acima:

1. **Verificar o navegador:** Tente em Chrome, Firefox, Edge ou Safari
2. **Console de erros:** Abra F12 → Console e procure por mensagens de erro
3. **Limpar cache:** Limpe o cache do navegador e recarregue
4. **Contato:** Entre em contato com o suporte via email
