Como Usar

🖱️ Com Mouse

Simplesmente clique nos botões das abas para navegar entre os conteúdos.

⌨️ Com Teclado

Use as setas esquerda ← e direita → do seu teclado para navegar entre as abas.

📱 Em Mobile

Toque nos botões das abas para trocar de conteúdo. O design se adapta automaticamente.

O sistema foi construído com foco em acessibilidade, suportando leitores de tela e navegação por teclado.
