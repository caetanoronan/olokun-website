# Pacote inicial de dados (Geomarketing Olokun)

Este diretório organiza os dados para análise multicritério (MCDA) de localização.

## Estrutura

- brutos/ — dados originais (IBGE, SIG, CSVs de parceiros)
- processados/ — camadas tratadas e padronizadas
- indicadores/ — tabelas finais com scores por setor/local
- metadados/ — dicionários, fontes e notas
- templates/ — modelos de planilhas e formulários

## Próximos passos

1. Baixar limites e setores censitários (IBGE Censo 2022).
2. Coletar pontos de parceiros (escolas, ONGs, empresas ESG).
3. Compilar camadas ambientais (APP, manguezais, nascentes).
4. Rodar processamento e gerar índices por área.
